## 📌 Python Productivity Hub

A menu-driven Python Productivity Hub that helps manage notes, tasks, expenses, and password strength checks. This project was built while learning Python modules, imports, functions, lists, dictionaries, and project organization.

## 🚀 Features
# 📝 Notes Manager
- Add notes
- View notes
- Delete notes
- Input validation for empty notes
  
## ✅ Task Manager
- Add tasks
- Mark tasks as completed
- View task status
- Delete tasks
  
## 💰 Expense Tracker
- Add expenses
- View expenses
- Delete expenses
- Calculate total expenses
- Validate numeric input
  
## 🔐 Password Checker
- Check password length
- Check for digits
- Check uppercase letters
- Check lowercase letters
- Check special characters
- Calculate password strength (Weak / Moderate / Strong)
  
## 🛠 Technologies Used
- Python 3
- Functions
- Modules
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Match-Case
- Input Validation
